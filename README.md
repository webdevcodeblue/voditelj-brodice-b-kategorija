# ⛵ Voditelj brodice B kategorije — aplikacija za učenje

Jednostavna web aplikacija (offline, jedna stranica) za pripremu ispita za **voditelja brodice kategorije B**. Sva pitanja i odgovori iz 5 područja na jednom mjestu, optimizirano za mobitel.

## Značajke

- **🎴 Kartice** — pitanje → otkrij odgovor → označi *Znam / Ne znam* (aktivno prisjećanje)
- **⚡ Sažeci** — brzo prelistavanje ključnih činjenica
- **🔍 Pretraživanje** po pojmu
- **📊 Praćenje napretka** — postotak po kategorijama (sprema se na uređaj)
- **Radi offline** — nakon otvaranja ne treba internet

## Područja (153 pitanja)

1. 🧭 Navigacija
2. ⚙️ Motoristika
3. 📜 Pomorstvo, propisi i meteorologija
4. ⚓ Manevriranje, sigurnost i prva pomoć
5. 📻 Radiotelefonija

## Kako koristiti

Otvori `index.html` u pregledniku (računalo ili mobitel). Na mobitelu: "Dodaj na početni zaslon" za rad kao aplikacija.

## Datoteke

- `index.html` — aplikacija
- `data.js` — pitanja i odgovori
